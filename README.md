# 杜娟医生官网静态版

这是一个可上线的静态官网雏形，包含首页、样式、官网视觉图、`robots.txt` 和 `sitemap.xml`。

上线前建议替换：

- `https://www.example.com/` 为正式域名
- 机构名称、城市、地址、电话、微信、小程序名称
- 医生真实执业信息、机构资质信息、专利证明展示
- 真实且合规授权的环境图、医生照和案例图

搜索引擎收录建议：

- 部署到正式域名并开启 HTTPS
- 在百度搜索资源平台提交站点和 `sitemap.xml`
- 每周更新 2-3 篇医美科普文章
- 页面避免使用“保证效果”“无风险”“最权威”等绝对化或承诺性表述
